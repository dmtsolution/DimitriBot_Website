# DimitriBot_Website

Bienvenue sur le projet **DimitriBot_Website**.

Ce projet est une plateforme dédiée au site touristique géré par DimitriBot.

## Description
DimitriBot_Website est une interface web permettant de présenter les attractions touristiques, gérer les réservations et fournir des informations utiles aux visiteurs.

## Fonctionnalités
- Présentation des points d'intérêt touristiques.
- Système de gestion de contenu.
- Interface utilisateur responsive.
